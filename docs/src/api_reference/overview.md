### Submodule Layers

- **`actool_rs::core`**: Low-level BOM, CAR, B-Tree, CSI, and Facet Hash16 implementations.
- **`actool_rs::codecs`**: LZFSE, CBCK, DMP2 Mini ISA, ASTC, and Ultra-HD spatial grid codecs.
- **`actool_rs::safety`**: ISO/CIE 11664-6 CIEDE2000, 80dB SNR HAS, and AutoSafe 4-gate barriers.
- **`actool_rs::assets`**: Sprite atlases, AppIcons, Layer Stacks, PBR 3D, and Audio assets.
- **`actool_rs::tools`**: Catalog compiler, CAREditor API, Mount system, and Repair engine.
